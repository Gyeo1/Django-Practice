# 장고와 리액트를 활용한 Instagram Clone 코딩


-기술 Stack: Django(Django-Rest-Framework), React, Docker, AWS     
-React로 Fornt-end 구현, Django로 Backend를 구현하였다.     
-추후 docker Image로 만들고 AWS Beanstalk에 배포까지 진행
